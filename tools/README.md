# External Tools

`DouyinLiveRecorder` 是本项目的外部直播录制依赖，不直接复制进仓库。

运行下面的命令会从官方 GitHub 仓库下载并固定到本项目验证过的提交：

```bash
npm run setup:local
```

固定版本：

```text
repository: https://github.com/ihmily/DouyinLiveRecorder.git
commit: add187f8d8c7ff7d231fcbee45cbb4f1ed247d3a
```

下载后的 `tools/DouyinLiveRecorder/` 和 Python 虚拟环境不会提交到本仓库。
